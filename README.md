### FishingMaster


#### 소개
낚시를 해서 도감을 채우는 간단한 낚시 게임

#### 조작
W/A/S/D : 이동
R : 도감 열기/닫기
F : 낚시 시작하기
Space : 낚시 할때 물고기 잡기, 대화창 상호작용
ESC : 대화창 닫기
